# AI-Assisted-Safety-Signal-Monitoring
## Overview
This project demonstrates an end-to-end safety analytics workflow using
SQL, Python, Power BI, and AI-assisted insights to support early signal detection
and patient safety monitoring.

## Tools Used
- SQL (PostgreSQL): Data extraction & validation
- Python (Pandas): Data quality checks & preprocessing
- Power BI: Visualization 
- AI Tools: Insight summarization and analytical support

## Note on Data
The dataset is not included due to size and data handling considerations.
All analysis logic is reproducible when the dataset is provided.

## Key Outcomes
- Identified drugs with higher adverse event frequency
- Compared Rx vs OTC safety patterns
- Assessed pregnancy category risk signals
- Generated AI-assisted analytical insights
